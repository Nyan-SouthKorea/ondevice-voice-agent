# LLM

이 디렉토리는 language model orchestration 계층 자리다.

현재 상태:

- 구현 시작 전
- wake word, VAD, STT 이후 들어오는 텍스트 명령을 해석하고 응답을 생성하는 레이어로 계획 중이다.

예상 역할:

- 명령 해석
- 대화 상태 관리
- 로봇 액션 또는 TTS 응답 생성과의 연결
