# LLM

이 디렉토리는 상위 대화 제어와 응답 생성을 담당할 language model 모듈 자리다.

## 목적

- STT 결과를 바탕으로 의도 해석
- 로봇 응답 텍스트 생성
- 필요 시 tool / action orchestration 확장

## 현재 상태

- 파일 구조만 준비됨
- 실제 구현은 아직 시작하지 않음
