  - wake word GUI 기본 대기 상태
- `wake_word_gui_detected.png`
  - `하이 포포` 감지 직후 상태
- `vad_demo_idle_terminal.png`
  - VAD terminal demo 대기 상태
- `vad_demo_speech_terminal.png`
  - VAD terminal demo 발화 감지 상태
- `vad_gui_idle.png`
  - VAD GUI demo 대기 상태
- `vad_gui_speaking.png`
  - VAD GUI demo 발화 감지 상태
