# Jetson Demo Videos

이 디렉토리는 Jetson에서 직접 실행한 데모 mp4 파일을 보관한다.
README에 바로 보이는 GIF 미리보기는 `../../gifs/jetson_demos/`에 별도로 정리한다.

파일 기준:

- `wake_word_gui_demo_jetson.mp4`
  - wake word GUI demo
  - `하이 포포` 호출 직후 감지 상태 확인용
- `vad_gui_demo_jetson.mp4`
  - VAD GUI demo
  - 발화 시 빨간 램프와 confidence 변화를 확인하는 영상

현재 영상은 아래 문서에서 직접 링크로 사용한다.

- `wake_word/README.md`
- `vad/README.md`
