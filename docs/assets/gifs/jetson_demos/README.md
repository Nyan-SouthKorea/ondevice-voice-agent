# Jetson Demo GIFs

이 디렉토리는 Jetson 데모 영상에서 추린 짧은 GIF 미리보기를 보관한다.

파일 기준:

- `wake_word_gui_demo_jetson.gif`
  - wake word GUI demo의 감지 구간 미리보기
  - README에서 클릭하면 원본 mp4가 열리도록 사용한다
- `vad_gui_demo_jetson.gif`
  - VAD GUI demo의 발화 구간 미리보기
  - README에서 클릭하면 원본 mp4가 열리도록 사용한다

현재 GIF는 아래 문서에서 직접 사용한다.

- `wake_word/README.md`
- `vad/README.md`
