# Archive

이 디렉토리는 활성 문서에서 내려온 상세 기록을 보관한다. 현재 기준 문서는 `docs/status.md`, `docs/개발방침.md`, `docs/decisions.md`, `docs/logbook.md`를 우선 본다.

## 현재 보관 문서

- `decisions_2026_03_full_before_refactor.md`
- `logbook_2026_03_full_before_refactor.md`

## 운영 규칙

- 활성 문서가 너무 길어지거나 현재 상태와 직접 관련 없는 이력이 많아지면 archive로 옮긴다.
- archive는 삭제 대신 보관을 위한 공간이다.
- archive 문서는 현재 기준 문서 역할을 대신하지 않는다.
