## 디렉토리 역할

- `보고서/`
  - 사람이 읽는 결과 요약, 구동 기록, 계획 문서
- `조사/`
  - 후보 모델 조사와 선택 배경
- `환경/`
  - A100, Jetson, 학습 환경 재현 절차

## 파일명 규칙

- 최신 기준으로 계속 유지할 문서는 고정 파일명을 쓴다.
  - 예: `README.md`
- 누적되는 조사/환경/보고서는 `YYMMDD_HHMM_한국어제목.md` 형식을 쓴다.
