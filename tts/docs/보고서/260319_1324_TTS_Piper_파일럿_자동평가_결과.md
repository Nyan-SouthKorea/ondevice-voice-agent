# TTS Piper 파일럿 자동평가 결과

> 마지막 업데이트: 2026-03-19
> 목적: `Piper` 한국어 pilot 학습 완료 후, 중요한 checkpoint들을 ONNX로 export하고 자동 benchmark 결과를 기록한다.

## 한 줄 결론

- `2.0시간 / 1,928문장` synthetic pilot로 `Piper medium scratch` 학습과 ONNX export, 자동 benchmark까지는 모두 성립했다.
- 하지만 현재 자동평가 결과는 품질이 아직 부족하다.
- `20`개 한국어 prompt 기준 `Whisper large-v3` 역전사 `Normalized CER`가 가장 좋은 checkpoint도 `0.9149` 수준이고, exact match는 전부 `0.0`이다.
- 따라서 지금 결과만으로는 full training으로 바로 확대하지 않고, synthetic data 품질/양/학습 설정을 다시 점검하는 쪽이 맞다.

## 현재 해석

- runtime 자체는 매우 빠르다.
  - mean RTF는 모든 checkpoint가 `0.1` 이하로, Jetson 후보 스택을 계속 검토할 가치는 있다.
- 그러나 자동 품질 지표는 아직 명확히 나쁘다.
  - 특히 `epoch 15` 이후부터는 CER가 더 크게 무너져 과적합 또는 teacher artifact 증폭을 의심할 수 있다.
- 자동지표 기준 추천 checkpoint는 `epoch=10-step=1298`다.
  - `mean_normalized_cer`가 가장 낮다.
- 다만 사용자가 실제로 듣고 다른 checkpoint를 고를 수 있도록, review sample은 계속 보관한다.
